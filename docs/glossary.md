# Glossary

#### frontier model

> These are ML models that span hundreds of billions to trillions of parameters
> or higher. These include large language models (LLMs) such as Gemini.

#### large model

> These are models that span tens to hundreds of billions of parameters or
> higher.

#### multi-host inference

> Inference with a model where the computational load is shared across multiple
> host machines.

#### single-host inference

> Inference with a model where the computational load is confined to a single
> host machine.
