# GKE Accelerated Platforms

## Base GKE Accelerated Platform

The Base GKE Accelerated Platform combines the essential components needed to construct a robust and adaptable platform using Google Kubernetes Engine (GKE).
This foundation provides a starting point for organizations to leverage GKE's container orchestration capabilities and build a tailored platform that meets
their specific requirements for scalability, reliability, and manageability. By assembling these core building blocks, the Base GKE Accelerated Platform
streamlines the process of establishing a GKE environment and accelerates the development and deployment of containerized workloads.

For additional information, see [Base GKE Accelerated Platform](/platforms/gke/base/README.md)
