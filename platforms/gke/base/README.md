# Base GKE Accelerated Platform

The [Terraservices](https://www.hashicorp.com/resources/evolving-infrastructure-terraform-opencredo) pattern is used in the Base GKE Accelerated Platform to deploy a foundational, "base", platform that can be adapted for a variety of use cases.

## Core

The Base GKE Accelerated Platform's [Core](/platforms/gke/base/core) is a flexible and customizable platform built on industry best practices, designed to be adaptable and scalable to a wide range of use cases and requirements. Its modular architecture allows for easy integration of additional components and services, enabling organizations to tailor the platform to their specific needs and workloads.

## Shared Configuration

The [Shared Config](/platforms/gke/base/_shared_config) provides a standardized framework that enables Terraservices to efficiently share common variables and local configuration.

## Features

[Features](/platforms/gke/base/features) are optional Terraservices that can be incorporated into the platform to further enhance the architecture.

## Use cases

The [use cases](/platforms/gke/base/use-cases) are a set of examples that utilize the base platform as their foundation.
